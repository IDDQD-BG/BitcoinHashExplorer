📘 Bitcoin Hash Explorer
Tools and analysis for exploring Bitcoin’s astronomical, mathematical, and cryptographic structure
📄 Scientific Publication
This repository accompanies the research preprint:

Lyubomir Stankov. Bitcoin’s Astronomical Architecture: Mathematical Invariants, Calendar Symmetries, and the Fibonacci–Fermat Asymptotic Bridge. 2026.  
HAL: https://hal.science/hal-05602626

🚀 Overview
Bitcoin Hash Explorer is a collection of tools, scripts, and mathematical models designed to analyze:

Bitcoin block hashes

Calendar‑based symmetries

Fibonacci and Fermat structures

Astronomical periodicities

Hash‑derived invariants

Non‑linear patterns in block generation

Този проект служи като изчислителна и визуална основа към научната статия.

🧠 Features
Hash pattern extraction

Block‑level statistical analysis

Time‑series exploration

Mathematical invariants

Visualization utilities

Reproducible research scripts
